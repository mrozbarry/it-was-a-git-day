# What is this?

It's a fun interface for teams (or individuals) to see and hear a custom live-stream of what's happening in their repos.

# Instructions

## Install

```bash
$ npm install
```

## Running

Run the development server

```bash
$ npm start
```

Build production javascript bundle

```bash
$ npm run compile
$ npm start
```

## Unit Testing

```bash
npm test
```
